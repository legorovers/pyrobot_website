Liverpool Python Robotics Resources

The Liverpool Python Robotics Resources (Pybots) consist of a set of worksheets that teach the Python Programming Language. These worksheets form a set of exercises that take the learner through the key features of Python all of which involve the control of a small robot. At present the resources support two robots that can be controlled using Raspberry Pis (The Initio and the Pi2Go). The resources also include simulated versions of each of these robots so it is possible to work through the exercises without owning or using a physical robot.
